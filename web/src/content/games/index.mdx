Welcome to the Casablanca Treasure Hunt!

This is a treasure hunt style game set in our beloved Casablanca. The goal is to explore the city while having fun.

At each step, you will be given a question along with different hints. Once you find the answer, enter it in the input box. If your answer is correct, you move on to the next step.
